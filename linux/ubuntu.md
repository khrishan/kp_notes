# Ubuntu Cheat Sheet

## Cerebo Search
```
sudo apt install gdebi

wget https://github.com/KELiON/cerebro/releases/download/v0.3.0/cerebro_0.3.0_amd64.deb

sudo gdebi cerebro_0.3.0_amd64.deb

```
